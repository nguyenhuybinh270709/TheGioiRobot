# The Gioi Robot

### An UI web for TheGioiRobot company.

## Tech Stack

- React
- TailwindCSS
- Other react library

## How to run project

### 1. Clone the project

```bash
git clone https://github.com/nguyenhuybinh270709/TheGioiRobot.git
cd TheGioiRobot
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run project

```bash
npm run dev
```
